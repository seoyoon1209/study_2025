자바스크립트 학습

이 가이드는 자바스크립트의 핵심 기본 문법을 학습하기 위해 제작되었습니다. 각 섹션의 코드 예제는 '실행' 버튼을 통해 직접 결과를 확인할 수 있습니다. 스크롤을 내리거나 왼쪽의 네비게이션을 클릭하여 학습을 시작하세요.

1. 변수와 데이터 타입

변수는 값을 저장하는 공간입니다. `let`, `const` 키워드를 사용해 선언합니다. 이 섹션에서는 변수 선언과 자바스크립트의 기본 데이터 타입에 대해 알아봅니다. 아래 코드의 실행 버튼을 눌러 각 변수의 타입을 직접 확인해 보세요.

let myName = "홍길동";
const myAge = 25;
let isStudent = true;

console.log("myName의 타입:", typeof myName);
console.log("myAge의 타입:", typeof myAge);
console.log("isStudent의 타입:", typeof isStudent);

연산자는 값에 대한 특정 연산을 수행합니다. 자바스크립트에는 다양한 연산자가 있으며, 특히 `==`(동등 연산자)와 `===`(일치 연산자)의 차이를 이해하는 것이 중요합니다. 아래의 비교 도구를 사용하여 두 연산자의 차이점을 직접 체험해 보세요.

3. 조건문

조건문은 특정 조건이 참인지 거짓인지에 따라 다른 코드를 실행하게 해줍니다. `if`, `else if`, `else` 구문이 대표적입니다. 아래 예제에서 `score` 변수의 값을 바꿔보고 실행하여 결과가 어떻게 달라지는지 확인해 보세요.

let score = 85;

if (score >= 90) {
    console.log("A 학점입니다.");
} else if (score >= 80) {
    console.log("B 학점입니다.");
} else {
    console.log("C 학점 이하입니다.");
}

4. 반복문

반복문은 특정 코드를 여러 번 실행해야 할 때 유용합니다. `for`문과 `while`문이 가장 일반적으로 사용됩니다. 각 반복문이 어떻게 동작하는지 아래 예제를 실행하여 확인하세요.

`for` 문

for (let i = 0; i < 5; i++) {
    console.log("반복 횟수:", i);
}

`while` 문

let i = 0;
while (i < 3) {
    console.log("현재 i:", i);
    i++;
}

5. 함수

함수는 특정 작업을 수행하는 코드 블록으로, 코드의 재사용성을 높여줍니다. 함수 선언식, 함수 표현식, 그리고 ES6에서 도입된 화살표 함수 등 다양한 방식으로 함수를 정의할 수 있습니다. 아래 예제에서 이름을 바꿔 함수를 호출하고 결과를 확인해 보세요.

함수 선언식

function greet(name) {
    return "안녕하세요, " + name + "님!";
}

let message = greet("김철수");
console.log(message);

6. 객체와 배열

객체와 배열은 여러 데이터를 구조적으로 관리할 수 있게 해주는 중요한 데이터 타입입니다. 객체는 키-값 쌍으로, 배열은 순서가 있는 목록으로 데이터를 저장합니다. 아래 예제를 실행하여 객체와 배열의 데이터를 어떻게 다루는지 확인해 보세요.

객체 (Object)

const person = {
    name: "Jane",
    age: 28,
    isMarried: false
};

console.log("이름:", person.name);
console.log("나이:", person['age']);

배열 (Array)

const fruits = ["사과", "바나나", "오렌지"];

console.log("첫 번째 과일:", fruits[0]);
console.log("과일 개수:", fruits.length);

fruits.push("포도");
console.log("추가 후 배열:", fruits);
