자바 기본 문법 개요

이 대화형 가이드는 자바 프로그래밍의 핵심 기본 문법을 소개합니다. 왼쪽 메뉴를 클릭하여 각 주제를 탐색하고, 코드 예제를 직접 실행해보며 개념을 익힐 수 있습니다.

자바는 **객체지향 프로그래밍 언어**로, 배우기 쉽고 플랫폼에 독립적이라는 강력한 장점을 가지고 있습니다. 모든 자바 코드는 **클래스(Class)**라는 단위 안에 작성되며, 프로그램의 실행은 항상 `main` 메서드에서 시작됩니다.

각 섹션에서는 설명과 함께 실행 가능한 코드 블록이 제공됩니다. '코드 실행' 버튼을 눌러 가상 콘솔에서 결과를 즉시 확인해보세요. 이를 통해 코드의 작동 방식을 직관적으로 이해할 수 있습니다.

1. 변수 (Variables)

변수는 데이터를 저장하기 위한 메모리 공간입니다. 자바에서는 변수를 사용하기 전에 반드시 어떤 종류의 데이터를 저장할 것인지 **타입(type)**을 명시해야 합니다. 이 섹션에서는 자바의 기본 데이터 타입과 참조 데이터 타입에 대해 알아보고, 실제 코드 예제를 통해 변수를 선언하고 사용하는 방법을 확인합니다.

데이터 타입

기본 데이터 타입 (Primitive)

  • int: 정수 (예: 10, -5)
  • double: 실수 (예: 3.14)
  • boolean: 논리값 (true 또는 false)
  • char: 단일 문자 (예: 'A')

참조 데이터 타입 (Reference)

  • String: 문자열 (예: "안녕하세요")

예제 코드

public class VariableExample {
    public static void main(String[] args) {
        int age = 25;
        double weight = 68.5;
        String name = "김철수";
        boolean isStudent = true;

        System.out.println("이름: " + name);
        System.out.println("나이: " + age);
        System.out.println("체중: " + weight);
        System.out.println("학생인가요?: " + isStudent);
    }
}

2. 연산자 (Operators)

연산자는 변수나 값을 가지고 계산을 수행하는 특별한 기호입니다. 덧셈, 뺄셈과 같은 기본적인 산술 연산부터 값들을 비교하거나 논리적인 판단을 내리는 데 사용됩니다. 이 섹션에서는 자바에서 자주 사용되는 연산자들의 종류를 살펴보고, 코드를 통해 각 연산자가 어떻게 동작하는지 직접 확인해봅니다.

주요 연산자

산술 연산자

+, -, *, / (나누기), % (나머지)

비교 연산자

== (같다), != (다르다), >, <, >=, <=

논리 연산자

&& (AND), || (OR), ! (NOT)

예제 코드

public class OperatorExample {
    public static void main(String[] args) {
        int a = 10;
        int b = 3;

        System.out.println("a + b = " + (a + b));
        System.out.println("a / b = " + (a / b));
        System.out.println("a % b = " + (a % b));
        System.out.println("a > b = " + (a > b));
        System.out.println("a == 10 && b == 3 = " + (a == 10 && b == 3));
    }
}

3. 제어문 (Control Flow)

제어문은 프로그램의 실행 흐름을 원하는 대로 제어하는 문법입니다. 특정 조건이 참일 때만 코드를 실행하거나, 특정 코드를 여러 번 반복 실행하는 등의 작업을 할 수 있습니다. 제어문을 통해 프로그램은 더 유연하고 동적인 작업을 수행할 수 있게 됩니다.

3-1. 조건문 (if-else)

조건문은 주어진 조건식의 결과(true 또는 false)에 따라 실행할 코드를 결정합니다. if문은 가장 기본적인 조건문으로, '만약 ~라면 ~를 실행하라'는 의미를 가집니다.

점수(score)를 조절해보세요

슬라이더를 움직여 `score` 변수 값을 변경한 후, '코드 실행' 버튼을 눌러 결과가 어떻게 바뀌는지 확인하세요.

85

예제 코드

public class IfExample {
    public static void main(String[] args) {
        int score = 85; // 슬라이더 값으로 변경됩니다.

        if (score >= 90) {
            System.out.println("A 학점입니다.");
        } else if (score >= 80) {
            System.out.println("B 학점입니다.");
        } else {
            System.out.println("C 학점 이하입니다.");
        }
    }
}

3-2. 반복문 (for, while)

반복문은 특정 코드를 여러 번 반복해서 실행해야 할 때 사용합니다. for문은 주로 반복 횟수가 정해진 경우에, while문은 특정 조건이 만족되는 동안 계속 반복해야 할 때 유용합니다.

for 문 예제

public class ForExample {
    public static void main(String[] args) {
        for (int i = 0; i < 5; i++) {
            System.out.println("반복 횟수: " + (i + 1));
        }
    }
}

    public static void main(String[] args) {
        int count = 0;
        while (count < 3) {
            System.out.println("Count: " + count);
            count++;
        }

4. 메서드 (Methods)

메서드는 특정 기능을 수행하는 코드의 묶음입니다. 한번 만들어두면 필요할 때마다 호출하여 재사용할 수 있어 코드의 중복을 줄이고 구조를 명확하게 만들어줍니다. 이 섹션에서는 메서드를 정의하고 호출하는 기본적인 방법을 알아봅니다.

메서드 구조

public static int add(int a, int b)

  • public static: 접근 제어자와 키워드입니다.
  • int: **반환 타입**. 메서드가 실행된 후 돌려주는 값의 종류입니다.
  • add: **메서드 이름**.
  • (int a, int b): **매개변수(파라미터)**. 메서드가 작업을 수행하는 데 필요한 값을 외부에서 전달받는 통로입니다.

예제 코드

public class MethodExample {
    public static void main(String[] args) {
        int sum = add(5, 3);
        System.out.println("두 수의 합: " + sum);
    }

    // 두 정수를 더하는 메서드
    public static int add(int a, int b) {
        return a + b;
    }
}
